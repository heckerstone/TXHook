该软件适合在安卓8.0以上系统运行，理论支持安卓7.0以上，但是很多问题。加群交流问题吧！（702991373）

## 已实现功能

 - 获取ShareKey\PublicKey\D2\A2...

 - 主动拦截固定Ecdh密钥及版本

 - 对Jce\Protobuf的自动分析
 
 - 过滤抓包，支持高级过滤（长按抓包页面的搜索栏展示/隐藏图标）
 
## 功能计划

 - 与PC端主流工具对接

 - 一键合成对应语言协议源码

 - 劫持并篡改设备信息

 - 劫持TLV\MD5\TEA代码
